# Airline Reservation App

A simple airline reservation interface built using HTML, CSS, and JavaScript.

## Features
- One-way, Round-trip, and Multi-city trip selection
- Input validation (dates and locations)
- Dynamic flight addition for multi-city trips
- Runs locally in the browser using VS Code Live Server

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open the project folder in VS Code
2. Right-click `index.html`
3. Click **Open with Live Server**
